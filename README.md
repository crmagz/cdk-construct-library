# CDK Construct Library

AWS CDK construct library scaffolded with projen, TypeScript 5.9.3, ESLint, GTS,
Prettier, and ESM package metadata.

## Commands

- `npm run lint`
- `npm run format`
- `npm run build`
- `npm run clean`
- `npm run deploy`

`npm run deploy` builds the package and publishes the generated tarball from
`dist/js` to NPM.
